My Projects


Political Times


Ultimate Goal:

Help improve social media responsibilty and literacy with political leaning assessment tools.


Process:

REST APIs were used to scrape posts from social media sites of popular news networks. Due to time and data constraints, a pre-trained model was used to assess the political tendencies of each site. Of course, this created drawbacks of not having direct control over the model and weights. On the other hand, I could thus guarantee that my own biases did not affect the results. The plot.ly API was used to visualize the results stored in a SQL database.


Future:

Continue to do research to create dashboards or visualizations that improve social media literacy. Find ways to accumulate quality data. Involve experts to avoid biases during labeling. Develop custom model.


Skills Used:

Python
SQL



Log Manager


Goal:

Create fast and efficient error log manager to create excerpt list using timestamps, keywords, and categories.


Process:

A vector of structs was created to hold the log entries. After being read in, they were sorted by timestamp to optimize for retrieval and memory storage later on. Unordered maps of keywords and categories were also created to allow for quick search. Searches returned results that the user could choose to add to an excerpt list (deque).


Skills Used:

C++



Galaxy Conference Identity


Goal:

Develop logotype, identity language, and mobile application mockups for sustainability conference.


Process:

A logotype was first created, considering the goals of the conference and name associations. User interface elements were then developed using shades of the existing color palette and free-space guidelines. Finally, a hifi application mockup was created using an existing client-specified wireframe.
